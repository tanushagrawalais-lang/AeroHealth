# AeroHealth
AeroHealth: Your Personal Health CGPA
AeroHealth is a sleek, mobile-inspired health dashboard built to gamify wellness. Inspired by the clean aesthetics of Samsung Health and the performance-driven world of F1, this app converts your daily vitals into a Health Score out of 10.0, giving you a single metric to track your physical well-being.

🌟 Key Features
Health CGPA Dashboard: A real-time scoring system that averages your performance across steps, BMI, hydration, and nutrition.

Dynamic Water Tracker: Interactive water logging (250ml, 500ml, 1L) with a fluid-fill CSS animation.

Smart BMI Calculator: Instant weight-to-height analysis with feedback on your health category.

Integrated Calendar: Track your daily streaks and stay organized with a built-in wellness schedule.

Immersive Nature UI: A calming nature backdrop designed to reduce stress while you monitor your stats.

Responsive Web Design: Fully optimized for both desktop and mobile viewing.
Tech Stack
Frontend: HTML5, CSS3 (Custom Fluid Animations), JavaScript (ES6+).

Backend: Python (Flask/FastAPI).

Deployment: Google Cloud Run.

Version Control: GitHub.
Quick Start & Deployment
Local Setup
Clone the repo:

Bash
git clone https://github.com/YOUR_USERNAME/AeroHealth.git
Install dependencies:

Bash
pip install -r requirements.txt
Run the app:

Bash
python app.py
Google Cloud Deployment
This project is containerized for Google Cloud Run.

Connect this repo to your Google Cloud Console.

Ensure the Dockerfile is in the root directory.

Deploy using Cloud Build for automatic CI/CD.
